Git 地址怎么填?看例子 / 怎么建 GitHub 私有仓库

填法:「Git Server」填域名,「仓库(repo)名」填 用户名/仓库名,再选协议。拼出来的完整地址:

  • GitHub · ssh:
    git@github.com:你的用户名/dotfiles.git
  • GitHub · https:
    https://github.com/你的用户名/dotfiles.git
  • 自建 git server · ssh:
    git@gitea.example.com:你的用户名/dotfiles.git
  • 自建 git server · https:
    https://gitea.example.com/你的用户名/dotfiles.git

ssh 推荐配好 SSH key 后用(免输密码);https 在首次 push 时输用户名 + Personal Access Token。

在 GitHub 建一个 Private 仓库(点开看步骤)
  1. 登录 GitHub → 右上角 + → New repository。
  2. Repository name 填 dotfiles(名字随意,后面要对上)。
  3. 选 Private(关键 —— 不要选 Public)。
  4. 可勾上 Add a README file(方便后续 clone)。
  5. 点 Create repository。
  6. 回到本工具:Git Server 填 github.com,repo 名填 你的用户名/dotfiles,协议按你配的 key 选 ssh 或 https。

GitLab / Gitee / Gitea / 自建 Forgejo 步骤几乎一样:新建仓库时选 Private 即可;Server 域名换成对应的就行。
